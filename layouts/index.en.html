Unlimited Customization Options

From simple CSS adjustments to your own config structure – CometVisu grows with your requirements

1

Change Color Scheme

Customize the entire appearance with just a few CSS variables

:root {
  --primary-color: #ff6b35;
  --background: #1a1a2e;
  --surface: #16213e;
}
2

Use Existing Designs

Choose from various included designs

Tile Pure Metal Planet
3

Custom CSS

Custom styles for individual adjustments without code changes

.cv-switch.myCustomStyle {
  border-radius: 20px;
  box-shadow: 0 4px 15px rgba(0,0,0,0.3);
}
4

Custom Widgets & Plugins

Develop custom widgets for special requirements

  • Plugin system for external libraries
  • Lazy-loading for optimal performance
  • Qooxdoo Framework for OOP development
5

Custom Config Structure

For maximum flexibility: Develop your own structure like the Tile structure

  • Custom XML schema definition
  • Custom parsers and controllers
  • Completely standalone visualization
