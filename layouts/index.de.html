Die moderne Tile-Struktur

Responsives Kacheldesign mit Web Components – perfekt für Touch-Bedienung auf Tablets und Smartphones

Kachelbasiertes Layout

Flexible 3x3-Grid-Kacheln für übersichtliche Steuerung. Widgets können über mehrere Zellen verteilt werden.

Touch-optimiert

Perfekt für die Bedienung auf Wandpanels, Tablets und Smartphones konzipiert.

Modernes Design

Dunkles Theme mit sanften Farbverläufen und animierten Übergängen für ein edles Erscheinungsbild.

Web Components

Native Custom Elements wie <cv-switch> für maximale Erweiterbarkeit.

Unbegrenzte Anpassungsmöglichkeiten

Von einfachen CSS-Anpassungen bis hin zur eigenen Config-Struktur – CometVisu wächst mit deinen Anforderungen

1

Farbschema ändern

Mit wenigen CSS-Variablen das komplette Erscheinungsbild anpassen

:root {
  --primary-color: #ff6b35;
  --background: #1a1a2e;
  --surface: #16213e;
}
2

Vorhandene Designs nutzen

Wähle aus verschiedenen mitgelieferten Designs

Tile Pure Metal Planet
3

Custom CSS

Eigene Styles für individuelle Anpassungen ohne Code-Änderungen

.cv-switch.myCustomStyle {
  border-radius: 20px;
  box-shadow: 0 4px 15px rgba(0,0,0,0.3);
}
4

Eigene Widgets & Plugins

Entwickle eigene Widgets für spezielle Anforderungen

  • Plugin-System für externe Bibliotheken
  • Lazy-Loading für optimale Performance
  • Qooxdoo Framework für OOP-Entwicklung
5

Eigene Config-Struktur

Für maximale Flexibilität: Entwickle deine eigene Struktur wie die Tile-Struktur

  • Eigene XML-Schema-Definition
  • Eigene Parser und Controller
  • Vollständig eigenständige Visualisierung
